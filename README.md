# stronghold-rat

The Rat sound pack for [peon-ping](https://github.com/PeonPing/peon-ping) — voice lines from The Rat in Stronghold Crusader.

## Install

```bash
peon packs install stronghold-rat
```

## Extras

The `sounds/extras/` folder contains all remaining voice lines from The Rat that aren't mapped to a CESP category. They're included so anyone can remix or extend this pack without needing the original game files.
